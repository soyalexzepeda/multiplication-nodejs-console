# Comandos disponibles en la aplicación
```
     Options:
          --help     Show help                                             [boolean]
          --version  Show version number                                   [boolean]
     -b, --base     Es la base de la multiplicación             [number] [required]
     -l, --limit    Marca el limite de la multiplicación        [number] [required]
     -t, --table    Muestra la tabla por consola         [boolean] [default: false]
```
